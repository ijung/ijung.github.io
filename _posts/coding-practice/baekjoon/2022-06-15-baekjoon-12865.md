# 문제
## [12865 평범한 배낭](https://www.acmicpc.net/problem/12865)
## ![screencapture](/posts/coding-practice/baekjoon/screencapture-acmicpc-net-problem-12865.png)

# 풀이
## 1. 전제 조건
- 다이나믹 프로그래밍을 이용한다.
  - 수식  
$
P_{n, w} = 
\begin{cases}
\begin{cases}
V_i + P_{n-1,\;w-W_i},\;if\;P_{n-1,\;w} < V_i + P_{n-1,\;w-W_i}\\
P_{n-1,\;w},\;otherwise\\
\end{cases},\;if\;W_i \leq w\\
P_{n-1,\;w},\;otherwise\\
\end{cases}\\
\\\,\\
n: 담을\,수\,있는\,물건의\,수(담을\,수\,있는\,물건:\,0 \leq i \leq n-1)\\
w: 배낭의\,무게\,한도\\
i: 물건의\,번호(i = n-1)\\
\\\,\\
P_{n, w}: n개의\,물건을\,담을\,수,있고,\,무게\,한도가\,w인\,배낭의\,가치의\,최대\,합\\
V_i: i\,물건이\,가치\\
W_i: i\,물건의\,무게\\
$

## 2. Step by Step

# 답
## kotlin code
```kotlin
fun main() {
    val (numOfItems, maxWeight) = readln().split(" ").map { it.toInt() }
    val items = Array(numOfItems) { readln().split(" ").let { Item(it.first().toInt(), it.last().toInt()) } }
    val bag = Array(numOfItems + 1) { IntArray(maxWeight + 1) { 0 } }

    for(n in (1..numOfItems)) {
        for(w in (1..maxWeight)) {
            val i = n-1
            val item = items[i]
            bag[n][w] = if(w >= item.w) kotlin.math.max(bag[n-1][w-item.w] + item.v, bag[n-1][w]) else bag[n-1][w]
        }
    }
    
    println(bag.last().last())
}

data class Item(val w: Int, val v: Int)
```
